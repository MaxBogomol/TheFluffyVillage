Unfinished release

Wizard's Reborn uses the Fusion Connected Textures mod for some blocks, this dependency is optional.
Fusion project recently had structure rewritten, which is why some Java package paths are now different.
Fluffy Fur creates its own predicate for Fusion's connected textures, which means I'll have to update the library to make it work on the latest versions.
However, due to Fusion updates, some parameters and render types for some blocks in Wizard's Reborn were broken and missing textures were displayed.
Aaand I don't like this at all. And since Wizard's Reborn depends on Fluffy Fur, I've decided to release the unfinished Wizard's Reborn update version 0.3. This is what it is as of now.
Some content is unfinished, some is completed, and some is completed but unavailable in survival due to other content being unfinished.

I wanted this very large update to come out as planned and polished, but it took me a very long time to make and I was getting in the way of other projects, programming, engineering, and studying.
Because this update took so long to make, I felt guilty that people who are waiting can't get it.. and I think I'll feel better after this.

Please read the big warning in the changelog before updating the mod.
And.. uh.. enjoy?
